The client for Monobank Currency Rate API

The unofficial wrapper for Monobank Currency Rate API. You can choose an artifact that depends on your favorite JSON library:

  • Gson,
  • Jackson,
  • Jackson Jr.

The Monobank Currency Rate API is a public service and is free to use. Currency rates are updated every 5 minutes, and the API has a rate limit of once per minute. To avoid your client from being banned, consider caching the response.
